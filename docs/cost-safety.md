# Cost safety

This repository is designed for careful beginners.

## Rules for Book 1

1. Prefer low-risk services and short-lived resources.
2. Do not leave buckets or credentials around carelessly.
3. Do not run commands you do not understand.
4. Cleanup is part of the exercise, not optional afterthought.

## Early scope control

Book 1 is intentionally narrow:

- AWS CLI
- IAM basics
- S3 basics

No large architecture. No sprawling multi-service lab. No “click everything” approach.

## Before real AWS use

Confirm:

- which AWS account you are using
- whether billing alerts are set up
- which identity you are signed in as
- which region you are working in
- that you know how to remove what you created
